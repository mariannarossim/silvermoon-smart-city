Projeto final autoral, proposto e desenvolvido por mim, aplicado com a turma **INF11** do Curso Técnico de Informática da escola **Grau Técnico — BH Centro**. O projeto consiste na construção de uma simulação de cidade inteligente, com enfoque em tecnologia, sustentabilidade e participação dos moradores, utilizando HTML, CSS, JavaScript e banco de dados relacional.

Este repositório mantém a versão do código desenvolvida e mantida por mim, servindo também como registro do escopo completo do projeto e meio de divulgação.

---

## Sobre o projeto

A proposta integra conhecimentos de front-end, lógica de programação e banco de dados em um produto único e contextualizado. A cidade simulada permite que o usuário acompanhe indicadores em tempo real, realize investimentos em áreas estratégicas e visualize os impactos das decisões tomadas, conectando conceitos técnicos a aplicações práticas de tecnologia urbana.

---

## Objetivos

O projeto tem como objetivo desenvolver uma simulação de cidade inteligente que permita:

- Explorar o mapa da cidade e seus pontos principais
- Realizar investimentos em áreas estratégicas
- Acompanhar a evolução dos indicadores da cidade em tempo real

### Escopo definido

- Nome da cidade
- Necessidade principal a ser atendida
- Moeda própria
- Identidade visual e imagens dos pontos principais

---

## Estrutura do projeto

| Módulo | Descrição |
|--------|-----------|
| **Página inicial** | Apresentação do nome e descrição da cidade, com destaque visual do mapa |
| **Mapa da cidade** | Representação gráfica com pontos principais e informações interativas |
| **Sistema de investimentos** | Moeda própria, áreas de investimento, ranking de prioridades, histórico e saldo integrados ao banco de dados |
| **Dashboard de gestão** | Gráficos e indicadores que acompanham a evolução da cidade conforme os investimentos realizados |
| **Necessidade principal** | Página dedicada à explicação de como a cidade atende à necessidade escolhida |
| **Identidade visual** | Imagens, paleta de cores, logotipo e padrão estético |
| **Interatividade** | Elementos dinâmicos, botões, modais e atualização em tempo real |
| **Documentação** | Explicação das escolhas, fluxos, telas, funcionalidades, guia de usuário e estrutura do banco de dados |

---

## Funcionalidades complementares

Recursos adicionais que enriquecem a simulação:

- História da cidade e narrativa de origem
- Personagens moradores, com nome, profissão e expectativas
- Simulação de crises e soluções por meio de investimentos
- Soluções sustentáveis: energia solar, reciclagem, transporte limpo
- Banco de dados completo: moradores, saldo, investimentos, itens adquiridos e evolução dos indicadores

## Tecnologias utilizadas

| Tecnologia | Uso no projeto |
|------------|----------------|
| **HTML5** | Estruturação semântica das páginas |
| **CSS3** | Estilização, responsividade e identidade visual |
| **JavaScript** | Interatividade, lógica de negócio e atualização dinâmica do dashboard |
| **Chart.js** | Geração de gráficos para o dashboard de indicadores |
| **MySQL / SQLite** | Banco de dados relacional para persistência das informações |

## Como executar

```bash
# Clone o repositório
git clone https://github.com/mariannarossim/cidade-inteligente.git

# Acesse o diretório do projeto
cd cidade-inteligente
```

Para a execução completa, recomenda-se o uso de um servidor local (por exemplo, a extensão *Live Server* do Visual Studio Code) e a configuração do banco de dados conforme os scripts disponíveis em `database/`.
